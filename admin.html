Como publicar: edite os campos abaixo, clique em Baixar, e substitua os arquivos baixados em assets/js/ no repositório (commit + push). O GitHub Pages atualiza o site automaticamente. Nada é salvo no servidor — tudo roda no seu navegador.
O painel sempre carrega a última versão publicada dos dados (sem cache). Para continuar de uma versão salva no seu computador, use ⬆️ Importar arquivo e selecione um *-data.js. Nos campos de imagem, use 📷 Enviar imagem para subir uma foto do PC — ela é redimensionada e vira um arquivo em assets/img/. No Chrome/Edge, clique antes em 📁 Conectar pasta (escolha a pasta do projeto) para salvar automaticamente; em outros navegadores a imagem é baixada para você mover para assets/img/. Depois faça commit da imagem junto com o *-data.js.
