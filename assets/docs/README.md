# Documentos

Coloque aqui a **tabela oficial de arbitragem** em PDF, com este nome exato:

```
tabela-arbitragem-fpt-2026.pdf
```

O botão **"Ver tabela oficial FPT 2026 (PDF)"** em `arbitragem.html` aponta para
`assets/docs/tabela-arbitragem-fpt-2026.pdf`.

Para publicar: salve o PDF nesta pasta com esse nome, faça commit e push.
Se a tabela for atualizada (ex.: 2027), troque o arquivo e ajuste o link em `arbitragem.html`.
